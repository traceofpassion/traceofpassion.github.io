R로 블로그 만들기(4) - 구글 애널리틱스(GA)와 디스커스(disqus) 연동하기

11월 13, 2020 in 블로그, 커스터마이징

5단계. 구글 애널리틱스(GA) 연동하기

구글 애널리틱스(GA)란, 블로그를 방문하는 사람들의 수, 국가, PC/모바일 여부 등 다양한 트래픽 관리 기능을 갖추고 있는 구글의 툴입니다. 블로그의 흥망성쇠를 측정하는 좋은 지표랄까요. 오늘은 GA를 내 블로그에 연동하는 방법을 알아보겠습니다.

5-1.

  • Google Analytics에 접속하여 구글 계정으로 로그인합니다.

  • 측정 시작 버튼을 클릭합니다.

  • GA 계정 이름(ID)를 설정합니다.

  • 속성 이름을 설정합니다. 속성은 사이트를 관리하는 하나의 단위로 이해하면 될 것 같습니다. 블로그 관련 속성임을 알아보기 편하게 이름 짓습니다.
  • 시간과 통화를 대한민국으로 설정합니다.

  • 비즈니스 정보는 기호게 맞게 설정합니다.

5-2.

  • 데이터 스트림 설정을 통해 GA 연동에 필요한 ID를 부여받습니다.
  • 블로그는 웹 사이트에 해당되므로 웹을 클릭합니다.

  • Netlify를 통해 생성한 URL을 기입하고, 해당 스트림의 이름을 기호에 맞게 설정합니다.

  • 생성된 스트림의 측정 ID를 확인합니다. 이 ID를 복사합니다.

5-3.

googleAnalytics = "측정 ID"
  • 부여받은 ID를 config.toml의 googleAnalytics 파라미터로 지정합니다.
blogdown::build_site()
  • Console 창에 코드를 실행하여 블로그를 빌드합니다.
$ cd TestBlog # 본인의 레포지토리 경로

6단계. 디스커스(disqus) 연동하기

디스커스(disqus)란, 블로그 댓글 호스팅 서비스입니다. 쉽게 설명하면 내 블로그에 댓글창을 추가하여 방문자와 의사소통을 할 수 있게 해주는 서비스라고 할 수 있습니다. 디스커스를 내 블로그에 연동하는 방법을 알아보겠습니다.

6-1.

  • Disqus에 접속하여 회원가입합니다.

  • Get Started 버튼을 클릭합니다.

  • I want to install Disqus on my site 탭을 클릭합니다.

  • Website Name, Category는 기호에 맞게 설정합니다.
  • Language는 현재 한국어 지원이 되지 않으므로 English로 설정합니다.

  • Plan은 무료로 사용 가능한 Basic 플랜으로 구독합니다.

6-2.

  • 이 탭을 클릭합니다.Disqus는 Hugo를 지원하지 않아서 코드를 직접 삽입해야 합니다.

  • 1번의 내용을 전체 복사합니다.

6-3.

  • ./themes/hugo-tranquilpeak-theme/layouts/partials/post/disqus.html를 메모장을 연결 프로그램으로 하여 오픈합니다.
  • 6-2에서 복사한 내용을 바꿔서 붙여넣기합니다.

R로 블로그 만들기(6) - 폰트 변경하기

2020년 11월 16일 00:00 in 블로그, 커스터마이징

블로그의 폰트를 설정하는 방법을 소개합니다.

9단계. 폰트 변경하기

blogdown에서 제공하는 기본 테마(hugo-lithium)에서 폰트 변경은 크게 어렵지 않습니다. 하지만, tranquilpeak 테마는 다릅니다. 사이트의 구조가 복잡한 편이라 그런지 폰트 변경이 까다롭습니다. 본 게시글은 HUGO로 사이트를 빌드하신 분들의 사이트를 참고하여 제작하였습니다.

9-1.

먼저 가장 상위 폴더로부터 위치한 ./themes/hugo-tranquilpeak-theme/src/scss/utils 경로를 탐색합니다. 두 가지 파일 _fonts.scss와 _variables.scss를 수정합니다. 각각 사용할 폰트를 웹으로부터 다운받는 역할과 폰트를 적용할 요소를 설정하는 역할을 합니다. 두 파일을 메모장을 연결 프로그램하여 오픈합니다.

9-2.

폰트를 정합니다. 저는 게시글에는 가독성이 좋으며 한글과 영문 적용이 동시에 되는 Noto Sans KR, 코드에는 네이버에서 배포한 나눔고딕코딩을 적용하겠습니다.

  • 위와 같이 설정한 후, @import 아래의 url을 복사합니다.
@import url('https://fonts.googleapis.com/css2?family=Nanum+Gothic+Coding:wght@400;700&family=Noto+Sans+KR:wght@100;300;400;500;700;900&display=swap')

9-3.

_fonts.scss를 다음과 같이 수정합니다.

# 기존
@font-face {
    font-family: 'Open Sans';
    font-style:  normal;
    font-weight: 400;
    src:         local('Open Sans'), local('OpenSans'), url(https://fonts.gstatic.com/s/opensans/v10/cJZKeOuBrn4kERxqtaUH3SZ2oysoEQEeKwjgmXLRnTc.ttf) format('truetype');
}

@font-face {
    font-family: 'Open Sans';
    font-style:  normal;
    font-weight: 700;
    src:         local('Open Sans Bold'), local('OpenSans-Bold'), url(https://fonts.gstatic.com/s/opensans/v10/k3k702ZOKiLJc3WVjuplzJS3E-kSBmtLoNJPDtbj2Pk.ttf) format('truetype');
}

@import url('https://fonts.googleapis.com/css?family=Merriweather:300,300i,400,700,700i&subset=cyrillic,cyrillic-ext,latin-ext');

# 추가
@import url('https://fonts.googleapis.com/css2?family=Nanum+Gothic+Coding:wght@400;700&family=Noto+Sans+KR:wght@100;300;400;500;700;900&display=swap');

9-4.

_variables.scss를 다음과 같이 수정합니다.

// Font families
$open-sans:            'Open Sans';
$open-sans-sans-serif: 'Open Sans', sans-serif;
$merriweather-serif:   'Merriweather', serif;
$menlo:                 Menlo, Consolas, monospace;
$noto-sans-kr: 'Noto Sans KR', sans-serif; # Noto Sans KR 폰트 추가
$nanum-gothic-coding: 'Nanum Gothic Coding', monospace; # 나눔고딕코딩 폰트 추가가

$font-family-base: $noto-sans-kr;

$font-families: (
   // base
   'headings':          $noto-sans-kr,
   // components
   'code':              $nanum-gothic-coding,
   'caption':           $noto-sans-kr,
   'image-gallery':     $noto-sans-kr,
   'post-header-cover': $noto-sans-kr,
   'post-meta':         $noto-sans-kr,
   'post-content':      $noto-sans-kr,
   'post-excerpt-link': $noto-sans-kr,
   'highlight':         $nanum-gothic-coding,
   // layout
   'sidebar':           $noto-sans-kr
);

9-5.

가장 까다로운 파트입니다. 해당 테마는 폰트 변경을 위해서는 npm의 grunt를 통해 빌드하는 과정을 담고 있습니다. npm은 Node.js의 기본 패키지 관리자입니다. Python의 pip 개념으로 이해하면 된다고 합니다. grunt는 태스크를 자동으로 수행하는 도구로써 Gruntfile 파일에 정의된 태스크를 수행합니다. grunt는 npm을 통해 배포됩니다. 폰트 변경을 반영하기 위해서는 npm을 설치하고, grunt를 설치하고, Gruntfile 파일에 정의된 태스크를 수행해야 합니다. 구체적인 과정은 다음과 같습니다.

  • Node.js 8.x 버전 설치하기
    • nodejs를 설치하면 npm이 함께 설치됩니다.
  • 설치된 Node.js command prompt를 실행합니다.
cd ./themes/hugo-tranquilpeak-theme/
npm install
npm install -g grunt-cli
grunt build
  • 순서대로 코드를 실행합니다.
  • 생성된 node_modules 폴더를 삭제합니다. css만 필요할 뿐, 빌드는 블로그다운으로 진행하기 때문입니다.

9-6.

blogdown::build_site()

$ cd TestBlog # 본인의 레포지토리 경로
$ git add .
$ git commit -m "fix fonts"
$ git push origin master

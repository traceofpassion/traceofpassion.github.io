R로 블로그 만들기(7) - LaTex로 수식 삽입하기

2020년 11월 18일 00:00 in 블로그, 커스터마이징

10단계. LaTex로 수식 삽입하기

데이터 사이언스 블로그를 운영하다보면 아무래도 수식을 사용할 때가 많습니다. 통계부터 머신러닝까지 모두 근간을 이루는 수학이 우선이기 때문인데요. RMarkdown에서는 문서 조판에 사용되는 프로그램 LaTex로 $ $ 사이에 있는 텍스트를 문법에 따라 수식으로 변환합니다. 다만 웹 브라우저에서 LaTex을 실행하기 위해서는 JavaScript 라이브러리 MathJax를 별도로 연동해야합니다.

10-1.

<script src="//yihui.org/js/math-code.js"></script>
<script async
src="//cdnjs.cloudflare.com/ajax/libs/mathjax/2.7.5/MathJax.js?config=TeX-MML-AM_CHTML">
</script>
  • 상위 폴더로부터 ./themes/hugo-tranquilpeak-theme/layouts/partials/에 위치한 `script.html’를 메모장으로 켜서 위의 코드를 가장 아래에 추가합니다.

10-2.

  • config.toml을 실행합니다.
[params]
  MathJaxCDN = "//cdnjs.cloudflare.com/ajax/libs"
  MathJaxVersion = "2.7.5"
  • [params] 아래 부분에 다음과 같이 두 가지 파라미터를 설정합니다.
  • CDNJS의 CDN 방식으로 배포된 ajax를 사용합니다. 버전은 2.7.5를 선택했습니다.

10-3.

blogdown::build_site()

$ cd TestBlog # 본인의 레포지토리 경로
$ git add .
$ git commit -m "add mathjax"
$ git push origin master
  • 블로그를 빌드한 후, 지금까지의 내용을 커밋하여 origin(깃헙)의 master 브랜치에 푸시합니다.

10-4.

blogdown::new_post(title = 'mathjax-test', ext = '.Rmd')
  • MathJax가 잘 작동되는지 테스트해보겠습니다.
- 기댓값의 정의는 다음과 같습니다.
$$E(X) = \int_{-\infty}^{\infty} xf(x) \, dx$$
  • 수식 삽입의 정상 작동을 확인할 수 있습니다.

R로 블로그 만들기(9) - 구글과 네이버 검색에 내 블로그 노출시키기

2020년 11월 20일 00:00 in 블로그, 커스터마이징

12단계. 구글과 네이버 검색에 내 블로그 노출시키기

내 블로그를 구글에 노출시키기 위해서는 Google Search Console에, 네이버에 노출시키기 위해서는 Naver Search Advisor에 웹사이트 주소를 등록해야 합니다. 그 전에 설정할 두 가지 옵션은 robots.txt와 favicon입니다.

12-1. robots.txt

로봇 배제 표준은 웹사이트에 로봇이 접근하는 것을 방지하기 위한 규약으로, 일반적으로 접근 제한에 대한 설명을 robots.txt에 기술합니다.

User-agent: *
Allow: /
  • 모든 로봇에게 문서 접근을 허락하기 위해 위와 같이 기입하여 가장 상위 폴더로부터 ./static/ 경로에 robots.txt 파일로 저장합니다.
# robots.txt
enableRobotsTXT = true
  • config.toml 파일에 위의 코드를 추가하여 저장합니다.
  • 블로그를 빌드한 후, 지금까지의 내용을 커밋하여 origin(깃헙)의 master 브랜치에 푸시합니다.

12-2. favicon

12-2-1.

파비콘은 웹사이트를 대표하는 아이콘입니다. 네이버에서 지원하는 파일 형식은 ico이므로 이를 어떻게 제작하는 지 알아보도록 하겠습니다. 무료 아이콘을 다운로드할 수 있는 flaticon에서 마음에 드는 아이콘을 조회합니다. 저는 테스트 아이콘을 선택했습니다. 아이콘을 다운로드하고 파일 이름을 favicon으로, 확장자를 ico로 변경합니다. favicon.ico 파일을 12-1과 동일하게 ./static/ 경로에 저장합니다.

12-2-3.

favicon = "/favicon.ico"
  • config.toml의 favicon 파라미터를 위와 같이 지정합니다.
  • 블로그를 빌드한 후, 지금까지의 내용을 커밋하여 origin(깃헙)의 master 브랜치에 푸시합니다.

12-4. Google Search Console에 등록하기

  • 구글 계정으로 로그인합니다.
  • Google Search Console에 접속하여 시작하기 버튼을 누릅니다.
  • URL 접두어 속성 유형을 선택하고, 블로그 주소를 입력합니다.
  • Google Analytics 계정을 등록하여 소유권 인정을 받습니다. (GA 인증 바로가기)
  • 좌측의 Sitemap 버튼을 클릭하여 사이트맵 URL 입력창에 sitemap.xml을 입력하고 제출합니다.
  • 2~3일 후에 연동이 잘 되었는지 웹사이트 이름을 구글 검색창에 검색하여 확인합니다.

12-5. Naver Search Advisor에 등록하기

  • 네이버 계정으로 로그인합니다.
  • Naver Search Advisor에 접속하여 상단의 웹마스터 도구 버튼을 누릅니다.
  • URL 입력창에 내 블로그 주소를 입력하고 제출합니다.
  • HTML 파일 업로드를 선택하고 HTML 확인 파일을 다운로드 받습니다.
  • HTML 확인 파일을 ./static/ 경로에 저장합니다.
  • 블로그를 빌드한 후, 지금까지의 내용을 커밋하여 origin(깃헙)의 master 브랜치에 푸시합니다.
  • 확인 버튼을 누른 후 사이트 정상 등록을 확인합니다.
  • 2~3일 후에 연동이 잘 되었는지 웹사이트 이름을 네이버 검색창에 검색하여 확인합니다.
